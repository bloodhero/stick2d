# stick2d 游戏引擎
专注2d的游戏引擎。

依赖的库：
* entt
* ffmpeg
* glad
* glm
* imgui
* json
* nuklear
* openal-soft
* opengl 3.3
* SDL2
* stb_image


# 开始
推荐使用 Visual Studio 2019，本游戏主要发布在windos平台。

使用 git 克隆本仓库：

`git clone --recursive git@github.com:bloodhero/stick2d.git`

执行 scripts/Win-GenProjects.bat 生成工程文件

使用 vs2019 打开 stick2d.sln 即可。
